# OpenGL shenanigans

These are some experiments I'm making to learn OpenGL. I'm currently following the [learnopengl.com](learnopengl.com) book as my main learning tool.

## Screenshots

![square](media/screenshot.png)

## Building

`
mkdir build && cd build
cmake ..
make
`

or

`
./run.sh
`
## Running

`
./build/opengl
`

or

`
./run.sh
`

## Controls

- **esc**: quit
- **space**: switch between fill and wireframe modes
